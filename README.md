# Saudi Air Connect

Saudi Air Connect is a flight booking web application built with a modern tech stack.

## Technologies Used

- **Framework**: React 18
- **Build Tool**: Vite
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **UI Components**: shadcn/ui
- **Icons**: Lucide React
- **Animations**: Framer Motion / Tailwind Animate

## Getting Started

### Prerequisites

- Node.js (v18 or later)
- npm

### Installation

1. Clone the repository:
   ```bash
   git clone git@github.com:ramahartadi/saudi-air.git
   ```

2. Navigate to the project directory:
   ```bash
   cd saudi-air-connect
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

### Development

Start the development server:
```bash
npm run dev
```

The application will be available at `http://localhost:5173`.

### Production

To create an optimized production build:
```bash
npm run build
```

The output will be in the `dist` folder.
